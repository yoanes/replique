====== Theme Instructions =======

*==== Basic Settings ====*
1) Logo: Add Logo for theme.
2) Favicon: Add favicon for theme.
3) Copyright Text box: Add Copyright text for footer.
4) Custom CSS: Add custom css instead using style.css file.
5) Color Scheme: Change color of theme.

*=== Sections ====*
In this tab you can add as many as sections for the homepage only also you can change the existing text.

*=== Homepage Slider ====*
In this tab you can add image sliders for homepage you can also add title and description of each slider.

*=== Footer ===*
1) About Text : Add some text about author/website.
2) Testimonial: Add some testimonial text.
